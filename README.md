# Assistente Acadêmico UFBA (RAG)

Este projeto implementa um Assistente Acadêmico da UFBA utilizando
RAG (Retrieval-Augmented Generation) com modelos locais.

## Tecnologias
- Python
- LangChain
- FAISS
- Sentence-Transformers
- Flan-T5 (Google)
- PDFs institucionais da UFBA

## Instalação

1.Extrair o arquivo zip.

2.Abrir a pasta no VS Code.

3.criar o ambiente virtual: python -m venv venv

4.Ativar o ambiente: .\venv\Scripts\activate (no Windows).

5.Instalar as bibliotecas: pip install -r requirements.txt

6.Rodar o projeto: python src/app.py
